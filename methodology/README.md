# WCED Methodology

This directory contains the working methodology specification for
the War Carbon Emissions Dashboard.

## Files

- v1.0.5.tex — LaTeX source for the current methodology
- v1.0.5.pdf — Compiled PDF (the canonical reference)
- CHANGELOG.md — Version history with diffs

## Status

This is a working draft documenting current pipeline implementation.
It is not a peer-reviewed publication. An academic preprint with
motivation, related work, and results is in preparation.
